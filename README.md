# Portfolio Project

A professional and modern portfolio website developed for Satish Yadav, focusing on a premium user experience with glassmorphism design principles and high performance.

## Core Features

- Modern Design System: Implements a premium frosted glass aesthetic for the navigation bar and interactive elements.
- Interactive Project Gallery: Distinctively styled project cards with rounded corners (2.5rem) and unique pastel color themes for both light and dark modes.
- Project Details Modal: Built with Radix UI to provide focused information about each project, including technologies, descriptions, source code links, and live demonstrations.
- Professional Experience: A structured timeline showcasing previous roles at ABO Infotech and Inkhub, with detailed achievement bullet points and company branding.
- Activity Integration: Real-time visualization of GitHub contribution activities through a dynamic chart.
- Global Theming: Full support for both light and dark modes with carefully curated color palettes to ensure high contrast and readability.
- Responsive Engineering: Optimized for a seamless experience across mobile, tablet, and desktop devices.
- Contact Infrastructure: Integrated contact form with backend notification logic.

## Technical Stack

- Frontend: Next.js 15, React 19, Tailwind CSS 4.
- UI Components: Radix UI, Lucide Icons, Magic UI.
- Animations: Framer Motion for smooth transitions and interactive feedback.
- Backend: Node.js, Express, Prisma ORM.
- State Management: Zustand.
- Deployment: Optimized for Vercel.

## Getting Started

1. Install dependencies:
   npm install

2. Configure environment variables:
   Create a .env file and add necessary backend configurations.

3. Run the development server:
   npm run dev

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
